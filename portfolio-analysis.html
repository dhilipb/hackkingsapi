Portfolio Analysis gives you data about a grouping of financial assets, for example performance, risk and asset allocation

Use cases: Asset Allocation is about diversifying your portfolio to include investments across stocks, fixed income and cash – so you don’t have all your eggs in one basket.

import java.net.*;
import java.io.*;

public class HackKingsAPIReader {
    public String readPortfolioAnalysis(String positions) throws Exception {
        String urlString = "https://test3.blackrock.com/tools/hackathon/portfolio-analysis";
        urlString += "?positions=" + positions;
        URL url = new URL(urlString);
        URLConnection urlConnection = url.openConnection();
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(
                                    urlConnection.getInputStream()));
        String inputLine;
        while ((inputLine = bufferedReader.readLine()) != null)
            System.out.println(inputLine);
        bufferedReader.close();
    }

    public static void main(String[] args) {
        HackKingsAPIReader apiReader = new HackKingsAPIReader();
        apiReader.readPortfolioAnalysis("BLK~25|AAPL~25|IXN~25|MALOX~25");
    }
}

API Documentation

Required Parameters
positions Specify the positions for which we are doing the analysis for. Identifiers and weights separated by '~' with individual positions separated by '|'. E.g.: BLK~25|AAPL~25|IXN~25|MALOX~25|
Optional Parameters
startDateA string that represents the end date of the analytics E.g.:20151212
endDateA string that represents the end date of the analytics E.g.:20151212
currencyA string that specifies the currency in which data will be returned E.g.: USD, GBP
calculateRiskAdds risk calculation to the output. E.g.: true, false
calculateExposuresChecks if it should calculates the exposure to the market risk. E.g.: true, false
calculateExpectedReturnsChecks if it should calculate the projected returns. E.g.: true, false
includeReturnsMapChecks if it should calculate the historic returns. E.g.: true, false
returnsTypeChecks if it should return DAILY or MONTHLY returns. E.g.: DAILY,MONTHLY
timeoutTime to wait before the requests times out. E.g.: 10000
