Portfolio Analysis gives you data about a grouping of financial assets, for example performance, risk and asset allocation

Use cases: Asset Allocation is about diversifying your portfolio to include investments across stocks, fixed income and cash – so you don’t have all your eggs in one basket.

import java.net.*;
import java.io.*;

public class HackKingsAPIReader {
    public String readData(String positions) throws Exception {
        String urlString = "https://test3.blackrock.com/tools/hackathon/portfolio-analysis";
        urlString += "?positions=" + positions;

        URL url = new URL(urlString);
        URLConnection urlConnection = url.openConnection();
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(urlConnection.getInputStream()));
        String inputLine;
        while ((inputLine = bufferedReader.readLine()) != null) {
            System.out.println(inputLine);
        }
        bufferedReader.close();
    }

    public static void main(String[] args) {
        HackKingsAPIReader apiReader = new HackKingsAPIReader();
        apiReader.readData("BLK~25|AAPL~25|IXN~25|MALOX~25");
    }
}

API Documentation

Required Parameters
positions Specify all the tickers with which you would like to build your portfolio. The total weight of the tickers should be 100. Identifiers and weights separated by ‘~’ with individual positions separated by ‘|’.
E.g.: BLK~25|AAPL~25|IXN~25|CASH-USD~25|
Optional Parameters
startDate A string that represents the end date of the analytics
E.g.: 20151212
endDate A string that represents the end date of the analytics
E.g.: 20151212
currency A string that specifies the currency in which data will be returned
E.g.: USD, GBP
calculateRisk Adds risk calculation to the output.
E.g.: true, false
calculateExposures Checks if it should calculate the exposure to the market risk.
E.g.: true, false
calculateExpectedReturns Checks if it should calculate the projected returns.
E.g.: true, false
includeReturnsMap Checks if it should calculate the historic returns.
E.g.: true, false
returnsType Checks if it should return DAILY or MONTHLY returns.
E.g.: DAILY,MONTHLY
timeout Time in milliseconds to wait before the requests times out.
E.g.: 10000
