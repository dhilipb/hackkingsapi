Performance Data captures the loss & gains of securities over a given period. Key performance data points include rates of returns, which is your profit on your investment over a specified period of time.

Use cases: Performance data gives you key insights into securities with strong and weak performance – as a portfolio manager you may want to see your top 10 performing assets over the past three months

import java.net.*;
import java.io.*;

public class HackKingsAPIReader {
    public String readSearchSecurities() throws Exception {
        String urlString = "https://test3.blackrock.com/tools/hackathon/performance";
        URL url = new URL(urlString);
        URLConnection urlConnection = url.openConnection();
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(
                                    urlConnection.getInputStream()));
        String inputLine;
        while ((inputLine = bufferedReader.readLine()) != null)
            System.out.println(inputLine);
        bufferedReader.close();
    }

    public static void main(String[] args) {
        HackKingsAPIReader apiReader = new HackKingsAPIReader();
        apiReader.readSearchSecurities();
    }
}

API Documentation

Required Parameters
identifiers Specify the securities for which you would like information.
E.g.: ‘IXN,GS’
Optional Parameters
includePositionReturns Checks if it should calculate the historic returns.
E.g.: true, false
returnsType Checks if it should return DAILY or MONTHLY returns
E.g.: DAILY,MONTHLY
startDate A string that represents the start date of the analytics.
E.g.: 20151212
endDate A string that represents the end date of the analytics.
E.g.: 20151212
