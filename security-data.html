Security Data concerns the ‘metadata’ associated individual financial securities. For example, BlackRock shares have data attributes like Country (US) & Currency (USD).

Use cases: Country code can be used to build a geographical map of your exposures to different countries across the globe. For example portfolio managers like to see their exposures to countries which are experiencing major political and economic events.

import java.net.*;
import java.io.*;

public class HackKingsAPIReader {
    public String readData() throws Exception {
        String urlString = "https://test3.blackrock.com/tools/hackathon/security-data";
        URL url = new URL(urlString);
        URLConnection urlConnection = url.openConnection();
        BufferedReader bufferedReader = new BufferedReader(new InputStreamReader(urlConnection.getInputStream()));
        String inputLine;
        while ((inputLine = bufferedReader.readLine()) != null) {
            System.out.println(inputLine);
        }
        bufferedReader.close();
    }

    public static void main(String[] args) {
        HackKingsAPIReader apiReader = new HackKingsAPIReader();
        apiReader.readData();
    }
}

API Documentation

Required Parameters
identifiers Specify the securities for which you would like information.
E.g.: ‘IXN,GS’
Optional Parameters
fromDate A string that represents the start date of the analytics
E.g.: 20151212
toDate A string that represents the end date of the analytics
E.g.: 20151212
